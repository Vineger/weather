# Hadoop--weather

- 使用 HDFS 进行hour, week, month的气象数据存储
- 使用 MapReduce 进行day, week, month的气象数据存储

# Point

- **file**: 指的是完整路径
- **fileName*: 仅代指文件名
- **path**: 文件所在的路径(即**file**包含**fileName**的)
